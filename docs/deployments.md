# Supported Deployments

`algo-pqc-kit` smart contract artifacts can be deployed to Algorand TestNet and MainNet.
Below are the reference app IDs for verification and testing.

## TestNet

| Contract Name | App ID | App Address | Explorer |
|---|---|---|---|
| FalconVault (Template) | `APP_ID_TBD` | `ADDRESS_TBD` | [Loras](https://loras.app/testnet) |
| PQCDao (Template) | `APP_ID_TBD` | `ADDRESS_TBD` | [Loras](https://loras.app/testnet) |

> **Note**: As we are in v0.1/alpha, the reference contracts on TestNet are regularly updated. 
> To test the Vault, use `PQCVault.deploy()` to launch your own isolated instance.

## MainNet

*MainNet deployments are pending the upcoming security audit in v0.4.*
